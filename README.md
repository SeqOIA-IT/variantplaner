A tool kit to manage many variant on desktop computer

## Installation

With `pip`:
```bash
pip install variantplanner
```

With [`pipx`](https://github.com/pipxproject/pipx):
```bash
python3.7 -m pip install --user pipx
pipx install variantplanner
```
